# Soccer API
Bakendi fyrir fótbolta gagnagrunn.

## Um verkefnið
Við unnum þetta verkefni sem hluti af Hugbúnaðarverkefni 1 (HBV501G) haustönnina 2025.
Verkefnið er unnið í Java Spring Boot.

## Til að keyra
Klónið verkefnið og keyrið `mvn spring-boot:run` á skipanalínunni
